## Project Overview

Educational Spyware Simulator demonstrates common behaviors used by spyware for cybersecurity learning.

Features include:

- System information collection
- Webcam video recording
- Microphone audio recording
- Audio-video synchronization
- Sending captured data through Telegram API (for demonstration)

This project helps students understand how spyware operates so they can analyze and detect such threats.
